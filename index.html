Introduction

Python is a versatile and powerful high-level programming language that emphasizes code readability and simplicity. Its clean and straightforward syntax allows beginners to quickly grasp programming concepts, while also offering advanced features that make it a preferred choice for experienced developers.

  • Python is known for its ability to integrate with other languages and tools, making it highly adaptable to various development needs.

Due to its flexibility, Python has become a popular language in many fields, including web development, data science, machine learning, artificial intelligence, scientific computing, automation, and software development.

  • Its vast standard library and large community support have helped it become one of the most widely used programming languages in the world.

Hello world

The "Hello, World!" program is a simple script that is often the first code written by beginners to learn the basics of a new programming language. It is a tradition across the programming community to start with this example, as it demonstrates how to display output to the user in a straightforward manner.

In Python, it's as simple as writing a single line of code:

print("Hello, World!")

Variables

Variables in Python are used to store information that your program can use and manipulate as it runs. They act as containers for data values, which can be changed or reused throughout your code.

  • The beauty of variables in Python is that they are dynamically typed, meaning you don't have to explicitly declare their data type; Python determines it automatically based on the assigned value.

In Python, you can create a variable by assigning a value to a name:

name = "Abel Gideon" age = 21

Control statements

Control statements in Python allow you to manage the flow of your program based on different conditions. These statements enable decision-making, looping, and branching in your code, making it possible to perform different actions depending on the circumstances or the values of variables.

  • Control statements are a fundamental aspect of programming as they give your code the power to adapt and respond to changing situations.

Python supports several control statements like if ... else statements and loops.

num = 10 if num > 10: print("Number greater than 10") else: print("Number is not greater than 10")
num = 1 while num <= 10: print(num) num += 1

Functions

Functions are one of the most essential building blocks in Python. They are reusable pieces of code designed to perform a specific task, which you can call whenever you need that task to be executed.

  • Functions help to break down complex problems into smaller, more manageable parts, making your code more organized, readable, and easier to debug or maintain.

In Python, you can define a function using:

def greet(): print("Hey!")
greet()
